### Add Some Content 
```bash
add an article [my-first-post.md] to folder [post]
$ hugo new post/my-first-post.md

add a new menu [about]
$ hugo new about.md
```


Edit the newly created content file if you want. Now, start the Hugo server with drafts enabled:
